Про сайт

📌 Описание проекта

Это проект для загрузки и валидации JSON файлов с сохранением данных в базу информации. Проект разработан для демонстрации навыков backend-разработки на Flask.

🛠️ Технологический стек
  • Python 3.8+ — язык программирования
  • Flask 3.0 — веб-фреймворк
  • Flask-SQLAlchemy 3.1 — ORM для работы с базой данных
  • Pydantic 2.x — Валидация и преобразование JSON-данных
  • SQLite — база данных (для тестов)
  • PostgreSQL — база данных (для production)
  • Bootstrap 5.3 — CSS фреймворк для UI
  • DataTables — плагин для таблиц с поиском и сортировкой
  • Jinja2 — Шаблонизатор для генерации HTML-страниц
📋 Функции сайта
  • Загрузка JSON файлов — форма для выбора и загрузки JSON файла с данными
  • Валидация данных — проверка формата JSON, длины имени (< 50 символов), формата даты (YYYY-MM-DD_HH:mm)
  • Сохранение в БД — сохранение валидных записей в PostgreSQL/SQLite таблицу
  • Просмотр записей — таблица с всеми записями с использованием DataTables (поиск, сортировка, pagination)
  • Обработка ошибок — уведомление пользователя о ошибках валидации с детальным описанием
📝 Формат JSON файла
[
    {
        "name": "random string less than 50 characters",
        "date": "YYYY-MM-DD_HH:mm"
    },
    ...
]
Правила валидации:
Правило Описание
JSON должен быть массивом Все данные в формате списка объектов
Ключи обязательны Наличие ключей name и date в каждом объекте
Длина name < 50 Имя должно быть строго меньше 50 символов
Формат даты Дата в формате YYYY-MM-DD_HH:mm (например: 2024-01-15_14:30)
Другие ключи Игнорируются, сохраняются только name и date
